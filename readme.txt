Standard Usage: Run src/phrase_unscrambler.py with no arguments and enter a scrambled phrase when prompted.

Supported Command Line Arguments:
-h, —-help            Help
-d, —-debug [level]   Show debug output at and below specified debug level
-t, --test            Run automated tests. See test/results.txt for results.


Debug Help:
Level 0 (default): No debug output
Level 1: Output Tokens
Level 2: Output all possible words
Level 3: Output all possible phrases
